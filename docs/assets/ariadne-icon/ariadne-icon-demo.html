Animation Specifications

Breathing Animation

Used for idle/active state. Provides subtle life without demanding attention.

  • Duration: 2 seconds
  • Easing: ease-in-out
  • Scale: 1.0 → 1.04 → 1.0
  • Glow: 2px soft → 6px strong → 2px soft

Whisper Ripple

Used for loading/thinking state. Barely visible ring that expands in sync with breathing.

  • Duration: 2 seconds (synced with breathe)
  • Easing: ease-in-out
  • Scale: 1.0 → 1.08 → 1.0
  • Opacity: 0.15 → 0 → 0.15
