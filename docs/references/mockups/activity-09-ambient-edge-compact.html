Concept: Brings back the ambient edge as a continuous 6px strip on the left. Each segment corresponds to an item in the list, creating a "heat map" you can see in peripheral vision. Gold = AI active, red = mention, blue = people, faded = some activity, invisible = quiet.

The ambient edge gives you:

  • • Peripheral awareness - see activity without looking directly
  • • Spatial memory - "the gold glow is near the top" = scratchpad
  • • Calm presence - no numbers, no badges, just color
  • • Scannable - glance at the edge, not the whole list
