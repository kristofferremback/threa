Nav 04: Label-Based Sidebar

User-defined labels group streams across types. A project like "Auth Project" can contain scratchpads, channels, threads, and DMs all together. The glow on the left indicates activity within the group.

How labels work:

  • • User creates labels (e.g., "Auth Project", "Q1 Launch")
  • • Any stream type can be added to a label
  • • Labels show aggregate activity glow
  • • Type badges show what kind of stream it is
  • • Uncategorized items fall to bottom
