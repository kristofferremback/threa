Nav 08: Accordion with Glow Summary

Collapsible accordion sections. Each section header shows a "glow summary" - mini pips indicating activity inside. Collapsed = see summary at a glance. Expanded = full list.

How it works:

  • • Glow pips in header = aggregate activity summary
  • • Gold = AI, Red = mention, Blue = people
  • • Click to expand/collapse sections
  • • User labels at bottom with same pattern
  • • One section open at a time (optional)
