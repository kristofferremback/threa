Activity 6: Compact Combined

Concept: Combines thread weight (left bar thickness) + presence glow (subtle gradient) + on-demand summaries (tooltip on hover). Compact 36px rows fit 30+ channels easily. Unread = bold name. AI/people/mention indicated by subtle presence dot.

Visual indicators:

  • • Thread weight (left bar) - thicker = more activity
  • • Presence glow (background gradient) - gold = AI, blue = people, red = mention
  • • Bold name = unread content
  • • Presence dot (right) - shows who's active
  • • Hover for summary - on-demand, not cluttering

Hover over items to see summaries

Summaries are lazy-loaded on hover. In production, could use intersection observer to pre-fetch as items come into view, or load on first hover and cache.
