Activity 7: Compact with Expandable Summaries

Concept: Same compact base as 06, but summaries expand inline on click instead of showing on hover. More intentional interaction, better for touch/accessibility, and allows the user to keep multiple summaries visible.

Interaction:

  • • Click any item with a chevron to expand its summary
  • • Click again to collapse
  • • Multiple items can be expanded simultaneously
  • • Items without summaries have no chevron

Why click instead of hover?

  • • Works on touch devices
  • • More intentional - user chooses what to see
  • • Can keep multiple summaries visible
  • • No accidental reveals while scrolling
  • • Easier keyboard navigation
