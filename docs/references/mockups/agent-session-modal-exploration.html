2. Modal Component Variants

Try different modal styles to see which feels best. Each shows the same streaming trace content.

Modal Variants:

1. Dialog (Centered)
• Traditional modal in center of screen
• Good for focused attention
• Familiar pattern

2. Drawer (Right Side)
• Slides in from right
• Doesn't obscure the conversation
• Similar to Slack threads

3. Sheet (Bottom)
• Slides up from bottom
• Mobile-friendly
• Less modal, more contextual
