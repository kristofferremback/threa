Activity 12: Uniform Height + Intense Glow

Concept: All items are the same 34px height. Priority is communicated through glow intensity - priority items have a wider, more saturated glow that spreads further into the sidebar. A small label provides context at a glance.

Glow intensity = importance:

  • • Priority items: 100px glow spread + box shadow
  • • Active items: 40px subtle glow
  • • Quiet items: no glow
  • • Small inline labels ("AI ready", "@you") for quick context

Why uniform height?

  • • Maximum density - fits more items in view
  • • The glow does the differentiation work
  • • Consistent rhythm is easier to scan
  • • Hover for full summary when needed
