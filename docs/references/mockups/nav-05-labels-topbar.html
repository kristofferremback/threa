Labels as Top Navigation

User-defined project labels become the primary navigation. Click a label to see all streams in that project. The glow under each tab shows aggregate activity.

Benefits:

  • • Project-focused rather than type-focused
  • • Full-width content area
  • • One click to see everything related to a project
  • • Activity glow aggregated per project
